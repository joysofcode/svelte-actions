## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/joysofcode/svelte-actions.git
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💿️ Run the development server

```sh
pnpm run dev
```
